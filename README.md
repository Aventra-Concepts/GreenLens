# GreenLens - AI Plant Identification & Care Platform

GreenLens is a production-grade web application that uses AI to identify plants and provide personalized care recommendations. Built with Next.js, TypeScript, and modern web technologies.

## üå± Features

- **AI-Powered Plant Identification**: Upload plant photos to get instant species identification using Plant.id API
- **Personalized Care Plans**: Get detailed care instructions generated by Google Gemini AI
- **Disease Detection**: Identify plant diseases and receive treatment recommendations
- **PDF Reports**: Download comprehensive care reports for your plants
- **My Garden**: Track and manage all your identified plants
- **Subscription Management**: Flexible pricing with multiple payment providers
- **Blog System**: Plant care tips and educational content

## üõ†Ô∏è Tech Stack

### Frontend
- **React 18** with TypeScript
- **Tailwind CSS** for styling
- **shadcn/ui** component library
- **Wouter** for routing
- **TanStack Query** for data fetching
- **React Hook Form** for form handling

### Backend
- **Express.js** with TypeScript
- **PostgreSQL** with **Prisma ORM**
- **Replit Authentication** via OpenID Connect
- **Multiple Payment Providers**: Stripe, Razorpay, Cashfree

### External APIs & Services
- **Plant.id v3** - Plant identification and health assessment
- **Google Gemini AI** - Care plan generation and disease advice
- **Perenual API** - Plant catalog information (primary)
- **Trefle API** - Plant catalog information (fallback)
- **Puppeteer** - PDF report generation
